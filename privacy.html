Privacy Policy

Effective Date: January 2026 | Last Updated: January 2026

1. Overview

This Privacy Policy describes how the TikTok OAuth Demo ("Application") collects, uses, and handles information. This is an educational project demonstrating OAuth 2.0 authorization flow.

2. Information We Collect

Authorization Code:

  • When you connect via TikTok, we receive an authorization code from TikTok
  • This code is stored locally in your browser (localStorage)
  • The code is NOT sent to external servers or databases
  • The code is temporary and expires in approximately 10 minutes

What We DON'T Collect:

  • We do not request access to your TikTok account details
  • We do not store user data on any server
  • We do not use analytics or tracking cookies
  • We do not share data with third parties

3. How We Use Information

The authorization code is used solely to demonstrate the OAuth authorization code flow. In a production application, this code would be exchanged on a backend server (using client_secret) for an access token to call TikTok APIs.

4. Data Storage

Browser Storage: Authorization codes are stored in your browser's localStorage. This data is:

  • Not sent to any external server
  • Cleared when you click "Clear Session"
  • Automatically cleared when you clear browser storage
  • Accessible only to this application domain

5. Security

This application uses:

  • HTTPS encryption for all communication with TikTok
  • State parameter validation to prevent CSRF attacks
  • No storage of sensitive credentials (client_secret is never in frontend code)
  • Secure OAuth 2.0 authorization code flow

6. Third-Party Services

This application uses TikTok OAuth for authentication. TikTok collects and processes information according to their Privacy Policy.

This application is hosted on GitHub Pages. GitHub may collect information as described in their Privacy Statement.

7. Your Rights

  • Access: You can view your authorization code while it's stored
  • Delete: You can clear session data anytime using the "Clear Session" button
  • No Tracking: We don't track your behavior across the web

8. Changes to This Policy

We may update this policy occasionally. Changes will be posted on this page with an updated "Last Updated" date.

9. Contact

For questions about this Privacy Policy or the application, please refer to the project GitHub repository.
